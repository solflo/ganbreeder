Ganbreeder is a collaborative art tool for discovering images. Images are 'bred' by having children, mixing with other images and being shared via their URL. This is an experiment in using breeding + sharing as methods of exploring high complexity spaces. GANs are simply the engine enabling this. Ganbreeder is very similar to, and named after, Picbreeder. It is also inspired by an earlier project of Joel Simon's, Facebook Graffiti, which demonstrated the creative capacity of crowds.

Ganbreeder uses DeepMind's BigGAN models, bundled to run fully offline.
